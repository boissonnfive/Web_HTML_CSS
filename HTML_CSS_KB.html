Créer un bouton

Démo

Code CSS


      .bouton {
      cursor: pointer;
      width: 120px;
      text-decoration: none;
      background-color: #000;
      border: none;
      color: white;
      padding: 15px;
      text-align: center;
      display: inline-block;
      font-size: 16px;
      margin: 4px 2px;
    }

Code HTML


    <button class="bouton" type="button">SUIVANT</button>
